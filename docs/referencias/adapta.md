ADAPTADOR WIRELESS DUAL BAND 2.4G / 5G 1300Mbps COM 2 ANTENAS

Este adaptador permite que seu dispositivo se conecte à rede Wi-Fi de 5GHz ou 2.4 GHz.

DUAL BAND 2.4GHz / 5GHz
Ideal para planos de internet de até 800 MEGAS utilizando a rede 5GHz.

APLICAÇÕES
Ideal para jogos online, assistir vídeos, filmes e séries em alta resolução, lives e vídeo conferências, sem travamentos. Também pode ser utilizado para trabalhar e estudar.

COMPATIBILIDADE
Funciona em qualquer computador, notebook ou tablet com sistemas Windows, atuando como uma placa de rede do equipamento. Basta plugar e começar a usar. Não é fornecido suporte para Linux.

COM ANTENAS
Equipado com 2 antenas para uma melhor recepção de sinal.

TECNOLOGIA REALTEK
Equipado com chipset Realtek 8812BU para uma conexão rápida e estável.

INTERFACE USB 3.0
Compatível com USB 2.0 e 1.1.

BOTÃO WPS
Permite conectar-se à rede Wi-Fi sem digitar senhas.

COMPATÍVEL COM HOTSPOT
Pode ser utilizado como um ponto de acesso à rede Wi-Fi.

FÁCIL INSTALAÇÃO
Plug and play em Windows 10. Para outros sistemas, é necessário instalar os drivers.
